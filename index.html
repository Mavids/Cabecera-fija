Backed, Frontend y Fullstack

De qué se trata Backed

El desarrollo backend es aquel que se encarga de diseñar la lógica y las soluciones para que todas las acciones solicitadas sean en una pagina web sean ejecutadas de manera correcta, por lo tanto un desarrollador backend es responsable de la programación del sitio en todos sus componentes; Podria decirse que es aquella persona que trabaja "detras del escenario" para mejorar la experiencia del usuario

Cómo lograr ser un Backed

Para poder ser un desarrollador Backend se deben tener unas clases de habilidades o se tienen que estar familiarizados con alguno de estos tips, los cuales son:

  • Saber las metodologías de programación
  • Conocer los fundamentos de la programación
  • Ser muy lógico y ordenado
  • Alta capacidad de concentración

    • Tambien puede ayudar tener conocimiento sobre los siguientes lenguajes de progración, los cuales son usados mucho al momento de convertirse en un desarrollador Backend

  • Python
  • Node.js
  • PHP
  • Go
  • Ruby
  • C#

De qué se trata Frontend

Es la parte de un programa o dispositivo a la que un usuario puede acceder directamente. estos Son todas las tecnologías de diseño y desarrollo web que corren en el navegador y que se encargan de la interactividad con los usuarios.

Cómo lograr ser un Frontend

Para convertirte en Frontend Developer debes saber HTML y CSS, los lenguajes de maquetación que nos permiten definir la estructura y estilos de una página web. Y también JavaScript, un lenguaje de programación para definir la lógica de nuestra aplicación, recibir las solicitudes de los usuarios y enviárselos al backend.

De qué se trata Fullstack

Es el habitante de los 2 mundos, del desarrollo front end y backend. Su tarea principal es tener conocimiento amplio tanto de front end como back end para apoyar en todo tipo de tarea, consulta o comunicación en el equipo de desarrollo.

Cómo lograr ser un Fullstack

Para ser considerado un desarrollador Fullstack, es necesario primero haber dominado el frontend o backend, posteriormente se adquiere el conocimiento restante con práctica y tiempo. Por ello es que un programador que tiene este tipo de puestos generalmente es llamado así porque supervisa, coordina y se encarga de todo el desarrollo web en general.
